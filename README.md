# Connect-4-Reinforcement-Learning

## Run
``` bash
# Play Game
python Connect_4/Game/Play.py
# Runs as a module
python -m Connect_4.test.test-winner-check
python -m Connect_4.test_test

# Frontend
cd ts-front
npm run dev

# Backend
cd django_back
python manage.py runserver 8888
```
